# Workout Tracker

A simple workout tracker web app for logging exercises, sets, reps, and weight.

## What it does

- Adds workout entries from a form
- Shows each exercise in a table
- Calculates total exercises, sets, and training volume
- Saves entries in the browser with local storage
- Lets you delete one entry or clear the whole workout

## How to open it

Open `index.html` in a web browser.

No installation is required because this app uses plain HTML, CSS, and JavaScript.
